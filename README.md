# Alish Joker Bot v.2
_May the Alish Joker Bot v.2 be with you..._

Tutorial
------
Recomended for python 2.7

Using git :

    $ git clone https://github.com/alishjoker/alishjokerbot


Python packages :

    $ pip install thrift==0.9.3
    $ pip install rsa
    $ pip install requests

Run Alish Joker Bot v.2 :

    $ python alish.py
